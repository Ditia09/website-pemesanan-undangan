Website Pemesanan Undangan merupakan aplikasi berbasis web yang digunakan untuk memesan undangan cetak maupun digital secara online. Sistem ini dibangun menggunakan PHP, MySQL, dan Bootstrap 5, serta dilengkapi dengan fitur perhitungan otomatis, diskon, dan integrasi WhatsApp.

Proyek ini bertujuan untuk:
1. Menerapkan pemrograman PHP prosedural.
2. Mengimplementasikan logika perhitungan bisnis (harga, jumlah, dan diskon).
3. Menghubungkan form HTML dengan PHP & MySQL.
4. Menerapkan best practices keamanan (Prepared Statement).
5. Mengintegrasikan WhatsApp API (wa.me) untuk pemesanan.

Teknologi yang Digunakan:
Frontend
- HTML5
- CSS3
- Bootstrap 5

Backend
- PHP 8+
- MySQL / MariaDB

Tools
- XAMPP / Laragon
- phpMyAdmin
- Browser (Chrome / Firefox)

Alur Pemesanan
User mengisi:
Nama
Nomor WhatsApp
Jenis Undangan
Jumlah Pesanan

Sistem menghitung:
Total harga
Diskon (otomatis)
Total bayar
Data disimpan ke database MySQL
Sistem mengarahkan ke WhatsApp admin dengan pesan otomatis
